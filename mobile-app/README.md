# Mobile Application

Mobile app cho khách hàng sử dụng FurniMart trên điện thoại.

## Tính năng

- Đăng nhập/Đăng ký
- Quản lý tài khoản và địa chỉ
- Tìm kiếm sản phẩm
- Xem chi tiết sản phẩm
- Xem sản phẩm 3D
- Thêm vào giỏ hàng
- Đặt hàng
- Theo dõi đơn hàng
- Đánh giá sản phẩm

## Công nghệ

- React Native hoặc Flutter
- 3D viewer (nếu hỗ trợ)
- Push notifications

## Cấu trúc

```
mobile-app/
├── src/
│   ├── screens/         # Screen components
│   ├── components/      # Reusable components
│   ├── services/        # API services
│   ├── navigation/     # Navigation setup
│   └── App.js
├── android/
├── ios/
└── package.json
```
